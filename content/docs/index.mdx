# Introduction

**Intentry** is an open-source tool for versioning, diffing, running, and collaborating on prompts.

Think Git for prompts — with a hosted platform, a public commons, and an open file format anyone can implement.

## What's here

- **CLI Reference** — the `intr` binary, all commands and flags
- **API Reference** — REST and GraphQL, auth, OpenAPI schemas
- **Concepts** — version store, diff engine, spaces, forks

Looking for the file format standard? See the [.prompt Spec](/spec).

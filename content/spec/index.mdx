# .prompt Specification

The `.prompt` file format is an open standard for defining versioned, executable prompts.
It extends [Dotprompt](https://google.github.io/dotprompt/) with version metadata,
diff support, and runtime contracts.

This specification is MIT-licensed. No Intentry account required to implement it.
